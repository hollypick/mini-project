# An interactive visualisation of Sanrio character popularity over time.

This project explores how Sanrio character popularity changes between 2020-2025, in 5 major countries. The goal is to make trends in popularity clear and easy to see and compare.
## How was the project made?

Built in R, the project utilised data obtained from a GitHub repository (which has since been deleted at the time of publishing) and transformed it into an interactive bar chart using the 'plotly' package.
