

# An interactive visualisation of Sanrio character popularity over time.

This project explores how Sanrio character popularity changes between
2020-2025, in 5 major countries. The goal was to make trends in
popularity clear and easy to see and compare between.

## How was the project made?

Built in R, the project utilised data obtained from a GitHub repository
(which has since been deleted at the time of publishing) and transformed
it into an interactive bar chart using the 'plotly' package.

## Dependencies - How to run.

The project was conducted on a Windows computer ( Windows version 23H2,
OS build 22631.6199 ), and made in Rstudio version 2025.05.1+513.

## Installing

The project can be found through [this
link](https://hollypick.github.io/mini-project/).

Required data is within the GitHub Repository
[mini-project/data](https://github.com/hollypick/mini-project/tree/main/data)
under the name 'sanriocharacterranking.xlsx'.

## Required packages

This project uses tidyverse, ggplot2, dplyr, ggalluvial, dplyr and
plotly.
